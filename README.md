Tool for encrypting and decrypting a file (to prevent it from being misused by
someone else) in DOS.

Its uses polygraphic substitution ciper for encryptions, and uses a checksum to
verify the integrity of the decrypted file. It can be used to prevent someone
else to misuse your file in case he/she gets hands on it. This is a DOS program,
hence you would need [DOSBox] to use it.
